========================================
  屏幕准心软件 - 部署说明
========================================

【必需文件】
  ✓ crosshair.exe     - 主程序（必需）
  ✓ resource/         - 准心图片文件夹（必需）
  ✓ bg.png            - 背景图片（推荐，否则使用默认渐变）

【可选文件】
  • logo.png          - 托盘图标（可选，否则使用默认图标）
  • 使用说明.md       - 使用说明文档（推荐）

========================================
  部署到其他电脑的步骤
========================================

方法1：完整复制
  1. 将整个 crosshair_release 文件夹复制到目标电脑
  2. 双击 crosshair.exe 运行

方法2：压缩包分发
  1. 将 crosshair_release 文件夹压缩为 zip
  2. 复制 zip 文件到目标电脑
  3. 解压 zip 文件
  4. 双击 crosshair.exe 运行

========================================
  文件结构示例
========================================

crosshair_release/
├── crosshair.exe          ← 主程序（3-4 MB）
├── bg.png                 ← 背景图片

========================================
  使用方法
========================================

1. 启动程序
   - 双击 crosshair.exe
   - 程序会最小化到系统托盘

2. 显示/隐藏准心
   - 快捷键: Ctrl + Shift + C
   - 或双击托盘图标

3. 打开设置窗口
   - 双击托盘图标

4. 退出程序
   - 右键托盘图标 → 退出

========================================
  注意事项
========================================

【系统要求】
  • Windows 7 / 8 / 10 / 11
  • 无需安装额外运行时环境
  • 静态编译，包含所有依赖

【文件大小】
  • crosshair.exe: 约 3-4 MB（静态编译）
  • resource 文件夹: 约 1-2 MB（准心图片）
  • 总大小: 约 5-6 MB

【杀毒软件】
  • 某些杀毒软件可能误报
  • 可添加到白名单

【游戏兼容性】
  • 建议游戏使用"无边框窗口模式"
  • 独占全屏模式可能无法显示准心

========================================
  常见问题
========================================

Q: 提示"找不到 resource 文件夹"？
A: 确保 resource 文件夹和 crosshair.exe 在同一目录

Q: 背景显示为渐变而非图片？
A: 确保 bg.png 和 crosshair.exe 在同一目录

Q: 准心不显示？
A: 1. 按 Ctrl+Shift+C 显示准心
   2. 检查游戏是否为独占全屏模式
   3. 切换游戏为无边框窗口模式
