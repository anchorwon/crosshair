## ✅ 程序已成功运行！

## 🎮 使用方法

### 1️⃣ 显示/隐藏准心

**三种方式**：

- **快捷键**（推荐）: 按 `Ctrl + Shift + C`
- **双击图标**: 双击系统托盘中的程序图标
- **菜单选择**: 右键托盘图标 → "显示/隐藏准心"

### 2️⃣ 更换准心样式

**方式1：预览窗口选择（推荐）** 🆕

1. **双击托盘图标**，打开准心预览窗口
2. 窗口中显示所有准心的缩略图，按分类排列
3. **点击任意准心**即可应用
4. 当前使用的准心会用绿色边框标记
5. 支持鼠标滚轮滚动浏览

**方式2：右键菜单选择**

1. 右键点击托盘图标
2. 选择 "选择准心样式"
3. 从分类菜单中选择准心

**准心分类**：
   - 🎯 丁字准心 (6种)
   - ➕ 十字准心 (8种)
   - 🔴 圆点准心 (7种)
   - ⭕ 小圆准心 (4种)
   - ⭕ 指示圈准心 (7种)
   - 🎖️ 战术准心 (1种)

### 3️⃣ 自定义准心颜色 🎨

**准心支持多种颜色**：

1. 右键点击托盘图标
2. 选择 "选择准心颜色"
3. 从颜色列表中选择：
   - ⚪ 原始颜色（不改变）
   - 🔴 红色
   - 🔵 蓝色  
   - 🟢 绿色
   - 🟡 黄色
   - 🟣 紫色
   - 🩵 青色
   - 🟠 橙色
   - 🩷 粉色

**特点**：
- 颜色立即应用到当前准心
- 预览窗口中也会实时显示颜色效果
- 切换准心后颜色保持不变

### 4️⃣ 退出程序

右键托盘图标 → 选择 "退出"

## ⚙️ 游戏设置建议

为了让准心在游戏中正常显示：

### ✅ 推荐设置

- **显示模式**: 无边框窗口模式（Borderless Windowed）
- **分辨率**: 与桌面分辨率相同
- **垂直同步**: 建议关闭（减少延迟）

### ❌ 不推荐设置

- **独占全屏模式** (Exclusive Fullscreen) - 准心可能无法显示

|------|----------|
| **VALORANT** | 设置 → 视频 → 显示模式 → 无边框窗口 |

| **Apex Legends** | 设置 → 视频 → 显示模式 → 窗口化 |

## ❓ 常见问题

### Q: 准心不显示？

**A**: 
1. 确保已按 `Ctrl+Shift+C` 显示准心
2. 检查游戏是否为独占全屏模式
3. 切换游戏为无边框窗口模式

### Q: 托盘图标没有显示我的 logo？

**A**: 
1. 确保 `logo.png` 在程序根目录
2. 检查文件名是否正确（区分大小写）
3. 重启程序

### Q: 快捷键不起作用？

**A**: 
1. 检查其他程序是否占用了 `Ctrl+Shift+C`
2. 尝试使用托盘菜单或双击图标
3. 以管理员身份运行程序

### Q: 准心偏移了？

**A**: 
- 重启程序，准心会自动居中

### Q: 游戏检测到外挂？

**A**: 
- 某些竞技游戏的反作弊可能检测该程序
- 请了解游戏规则，谨慎使用
- 可以在非竞技模式下使用

## ❓ 常见问题

### 🎮 全屏游戏中看不到准心？（重要）

**适用于**: CS 1.6/1.7、CS:GO、老游戏等使用独占全屏模式的游戏

**⚠️ 核心问题**: 
独占全屏模式（Exclusive Fullscreen）下，游戏直接控制显示器输出，**完全绕过Windows窗口系统**。这是**系统级别的硬性限制**，任何外部程序都无法在独占全屏游戏上显示窗口。

## 🎯 CS 1.6/1.7 准心解决方案

### ✅ 方案1：窗口化全屏模式（推荐，100%有效）

**Steam版本：**
1. 在Steam库中找到 Counter-Strike 1.6
2. 右键 → 属性 → 启动选项
3. 添加以下参数（复制粘贴）：
```bash
-window -noborder -w 1920 -h 1080 +exec autoexec.cfg
```
> 💡 将 `1920 -h 1080` 改为您的屏幕分辨率

**非Steam版本：**
1. 找到 `hl.exe` 或 `cstrike.exe` 快捷方式
2. 右键 → 属性 → 目标
3. 在目标路径最后添加（注意空格）：
```bash
-window -noborder -w 1920 -h 1080
```

**配置文件方式：**
1. 找到 `cstrike/config.cfg`（游戏目录下）
2. 用记事本打开，添加或修改：
```
_windowed_mouse "1"
vid_config_x "1920"
vid_config_y "1080"
```
3. 保存后重启游戏

**设置后效果：**
- ✅ 游戏全屏显示，无边框
- ✅ 外部准心程序可以正常显示
- ✅ 性能几乎无影响
- ✅ 可以快速Alt+Tab切换

---

### ✅ 方案2：使用CS内置准心（简单，无需外部程序）

CS 1.6 有强大的内置准心系统：

**快速设置：**
1. 游戏中按 `~` 键打开控制台
2. 复制粘贴以下命令：

```bash
// 推荐准心配置（复制粘贴）
cl_crosshair_color "0 255 0"        // 亮绿色准心
cl_crosshair_size "medium"          // 中等大小
cl_crosshair_translucent "0"       // 不透明
cl_dynamiccrosshair "0"            // 静态准心（不随移动变化）
```

**颜色选项：**
```bash
cl_crosshair_color "255 0 0"       // 红色
cl_crosshair_color "0 255 0"       // 绿色
cl_crosshair_color "255 255 0"     // 黄色
cl_crosshair_color "0 255 255"     // 青色
```

**大小选项：**
```bash
cl_crosshair_size "small"          // 小
cl_crosshair_size "medium"         // 中
cl_crosshair_size "large"          // 大
```

**保存设置：**
在控制台输入：
```bash
writeconfig
```

### 🔧 故障排除

**问题1：改成窗口模式后，游戏画面变小了**
- 确保启动参数中的分辨率与您的显示器分辨率一致
- 例如：2K显示器用 `-w 2560 -h 1440`

**问题2：Alt+Tab后准心消失**
- 按 `Ctrl+Shift+C` 重新显示准心
- 或者在托盘图标右键菜单中选择"显示准心"

**问题3：窗口化后游戏性能下降**
- 关闭系统的"游戏模式"
- 降低游戏内特效设置
- 现代电脑基本不会有明显性能差异

### 🖥️ 多显示器支持

准心会始终显示在**主显示器**的屏幕中心。如果在副显示器玩游戏，建议：
- 将游戏所在的显示器设置为主显示器
- 或使用游戏内准心功能
